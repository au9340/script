狂魔哥，本名张本逸涵（中文名刘斗江），是一位以王者荣耀游戏解说闻名的网络主播。
狂魔哥出生于2000年7月21日，在日本北海道长大，后来定居于中国浙江省温州市。他的直播主要在B站和抖音进行，以王者荣耀游戏解说为主，抖音账号“狂魔哥解说”粉丝超过77万，获赞超过420万。狂魔哥的直播风格独特，语速快、音量大，常以幽默甚至挑衅的方式与队友互动，这种风格吸引了大量观众，但也引发争议。 

游戏表现与争议
狂魔哥擅长模仿和学习其他国服玩家的出装和打法，形成自己独特的游戏风格。然而，他在线上经常被1级单杀，团队作战时有时表现不佳，例如偷吃队友资源或低参团率，却常以高语速压制队友，甚至在直播中邀功。此外，他在直播中发放的红包有时存在欺骗性，5元人民币红包被称作“100日元大红包”。 

发展历程
狂魔哥早期在B站直播并不出名，直到2021年左右签约“进击的金厂长”，才逐渐积累粉丝并走红。其火爆的方式包括点评人物黑历史和游戏事件，这也为他带来了大量关注和争议。后来，他从B站转到虎牙直播继续发展。 

个人背景
狂魔哥的父亲是驻日美军，母亲来自乌干达。他自幼在中国上学，因此能流利使用中文与观众交流。他的真实身份和背景在网络上曾引发讨论，但他本人多次强调自己是中国人。 

总的来说，狂魔哥是一位以高操作水平和争议性直播风格著称的王者荣耀主播，其独特的个性和直播方式使他在网络上拥有较高的知名度。
